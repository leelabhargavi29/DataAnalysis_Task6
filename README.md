Task 6: K-Nearest Neighbors (KNN)

Tools Used:
- Python
- Scikit-learn

Dataset:
- Iris Dataset

Steps:
1. Load the Iris dataset.
2. Split the data into training and testing sets.
3. Choose the K value.
4. Train the KNN model.
5. Predict classes for test data.
6. Evaluate the model using accuracy.

Result:
- Accuracy obtained using KNN model.

Interview Questions:

1.What is KNN?
A classification algorithm that predicts using the nearest neighbors.

2. How do you choose the value of K?
Test different K values and choose the one with the best accuracy.
